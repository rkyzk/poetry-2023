# Your Poetry

詩を書いてシェアするプラットフォームです。他のメンバーから感想、フィードバックをもらうことができます。プロフィールを作成し簡単に自己紹介もできます。

## このサイトの機能

**すべてのユーザー（未登録のサイト閲覧者を含む）ができること**

- ホーム画面閲覧
- おすすめの作品リストを閲覧
- 「過去２週間に投稿された詩」のリストを閲覧
- 「人気の詩」リストを閲覧
- カテゴリー別のリストを閲覧
- 詩を検索
- ユーザープロフィール閲覧、検索
- メンバー登録

**登録済みメンバーができること（上記に加えて）**

- ログイン、ログアウト
- 詩を投稿、更新、削除
- コメント投稿、更新、削除
- 自分のプロフィール更新
- 他ユーザーのプロフィールをフォロー
- 詩に「いいね」を追加
- 自分が書いた詩のリストを閲覧
- 「いいね」を追加した詩のリストを閲覧
- フォローしているプロフィールのリストを閲覧
